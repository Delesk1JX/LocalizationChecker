### 2. Система сортировки таблиц ✅

Реализирована интеллектуальная сортировка при клике на заголовки столбцов:

#### Поведение сортировки:

**Текстовые колонки (Мод, Причина):**
- 1-й клик: от A-Z (алфавитный порядок)
- 2-й клик: от Z-A (обратный порядок)

**Числовые колонки (%, Ключи RU, Ключи EN, Не хватает):**
- 1-й клик: от большего к меньшему (убыванию)
- 2-й клик: от меньшего к большему (возрастанию)

#### Примеры использования:

- **Нажимаем "%"** → Моды сортируются по проценту перевода (100% → 0%)
- **Нажимаем "%" еще раз** → Сортировка меняется на обратную (0% → 100%)
- **Нажимаем "Мод"** → Сортировка от A-Z
- **Нажимаем "Мод" еще раз** → Сортировка от Z-A
- **Нажимаем "Ключи RU"** → Сортировка по количеству ключей (больше → меньше)

### Как эта система работает 🔄

1. **Первый клик на новую колонку:**
   - Для числовых полей: начинает сортировку по убыванию
   - Для текстовых полей: начинает сортировку по возрастанию (A-Z)

2. **Повторный клик на ту же колонку:**
   - Переворачивается направление сортировки
   - Клик на "%" дважды: 100% → 0%, затем 0% → 100%

### Структура сортировки в коде 📝

```python
# Числовые колонки начинают с убыванию
numeric_columns = ["%", "Ключи RU", "Ключи EN", "Не хватает"]
self.sort_state[category]["reverse"] = column in numeric_columns
```

Это обеспечивает интуитивное поведение для пользователя:
- Хотите увидеть полностью переведенные моды вверху? Нажимайте "%" ✅
- Хотите увидеть моды с большим количеством ключей вверху? Нажимайте "Ключи EN" ✅
- Хотите увидеть моды в алфавитном порядке? Нажимайте "Мод" ✅
